下划线相关

如果要给一段文字加上下划线，第一个想到的是css样式text-decoration: underline。

This is underline






不过，也可以使用border-bottom来伪装成下划线。
>

This is span border-bottom

这样写的好处是，可以控制“下划线”的距离、高度（粗细）和颜色。

但是如果有多行都需要下划线，上面的第二种方法就不管用了，只会在最底下显示一行”下划线”。

This is span border-bottom.This is span border-bottom.This is span border-bottom.This is span border-bottom.This is span border-bottom.This is span border-bottom.This is span border-bottom.

而第一种方法text-decoration虽然管用，但是不能控制下划线的样式。

This is underline.This is underline.This is underline.This is underline.This is underline.This is underline.This is underline.This is underline.This is underline.This is underline.This is underline.This is underline.

所以，解决上面所有问题的方法是，先将一段文字设置属性display: inline，即：把它设置成行内元素，然后再给它加上底边border-bottom。

This is span border-bottom.This is span border-bottom.This is span border-bottom.This is span border-bottom.This is span border-bottom.This is span border-bottom.This is span border-bottom.

行高、距离等属性随后自由设置即可。
